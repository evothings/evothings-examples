This app automatically scans for BLE devices. Note that you must run this app in the EvoThings client app, or build a Cordova app with the plugin com.evothings.ble. Device with support for Bluetooth 4.0 and Android 4.3 or higher required.

How to use: The app should detect running BLE devices.

Troubleshooting: There are known problems with that BLE stops working on multiple reloads. In this case, 1) stop the EvoThings client app, 2) turn off and on your Bluetooth, and 3) start the EvoThings client app again. Additionally, you can use the EvoThings Workbench Tools to see any error messages that may help in your investigation.
